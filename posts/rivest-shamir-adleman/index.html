RSA(Rivest–Shamir–Adleman)란?

RSA 알고리즘에 대해 알아보자

  • RSA는 블록 암호 시스템의 일종으로, 비대칭키(공개키) 암호 알고리즘이다.

  • 1978년 Ronald Rivest(로널드 라이베스트), Adi Shamir(아디 샤미르), Leonard Adleman(레오나드 애들먼)가 개발했다.

  • 아주 큰 수를 소인수분해하기 어렵다는 점을 이용한 암호화 알고리즘이다.

  • 나중에 아주 큰 수의 소인수분해를 쉽고 빠르게 하는 방법이 생겨난다면 RSA는 무력화될 수 있다.

  • 일반적인 암호화 뿐만이 아닌 메시지 인증, 공인인증서, 키 교환 등 다양한 곳에 사용되고 있다.

RSA 암호화 방식#

  1. 소수인 $p$와 $q$ 생성(나눠지는 수가 1과 자기 자신밖에 없어야된다.)
  2. $n = p × q$
  3. $∅(n) = n~(n-1)$ 사이의 정수들의 갯수. 단, $p$와 $q$는 소수이므로 $∅(n) = (p - 1) × (q - 1)$
  4. $∅(n)$과 서로소(최대공약수가 1)이고, $1 < d < ∅(n)$인 $d$ 선택.
  5. $d$의 역원인 $e$ 계산. .
    • 공개키: ($e$, $n$)
    • 개인키: ($d$, $n$)
    • $e$를 개인키로, $d$를 공개키로 사용해도 상관 없다. 하지만, 더 큰 수를 개인키로 사용하는 것이 안전하다.
    • $C$: 암호문, $P$: 평문
    • 암호화: $C = P^{e} mod:n$
    • 복호화: $P = C^{d} mod:n$

  • 예제

    1. 소수 $p = 7$, $q = 17$ 생성

    2. $∅(n)$과 서로 소수이고, $1 < d < ∅(n)$인 d 선택. $d = 77$

    3. $d × e mod ∅(n) = 1$인 $e$ 선택($d$와 $∅(n)$의 역원 계산). $e = 5$

      • 공개키: (5, 119)

      • 개인키: (77, 119)

    4. 전송하고 싶은 데이터 선택: 5

    5. 데이터를 암호화: $(5 ^ 5):mod:119=31$

    6. 암호화된 데이터를 복호화: $(31^77):mod; 119 = 5$

Euclid Algorithm#

  • Euclid 알고리즘은 최대공약수를 알아내기 위한 알고리즘이다.

  • RSA의 키를 생성할때 두 수의 최대공약수가 1인 것을 찾아야 하기 때문에 이 알고리즘이 필요하다.

    int gcd(int a, int b) { 
    int tmpA = a; int tmpB = b; int remainder = 0;
    while(tmpB != 0) { 
        remainder = tmpA mod tmpB; 
        tmpA = tmpB;
        tmpB = remainder; 
    }

    return tmpA;
}

Extended Euclid Algorithm#

  • Euclid 알고리즘을 확장해서 최대공약수와 역원을 구하는 알고리즘이다.

    typedef struct __gcd {
  int gcd;
  int inverse;
} gcd;

gcd extGcd(int a, int b)
{
  gcd sRet = { -1, -1 };
  int A1 = 1;
  int A2 = 0;
  int A3 = a;
  int B1 = 0;
  int B2 = 1;
  int B3 = b;

  while (sRet.gcd == -1) {
    int Q, T1, T2, T3;

    // 최대공약수: A3, 역원: 없음
    if (B3 == 0) {
      sRet.gcd = A3;
      sRet.inverse = 0;
    }
    // 최대공약수: B3, 역원: B1
    else if (B3 == 1) {
      sRet.gcd = B3;
      sRet.inverse = B2;
    }

    Q = A3 / B3;
    T1 = A1 - Q * B1;
    T2 = A2 - Q * B2;
    T3 = A3 - Q * B3;
    A1 = B1;
    A2 = B2;
    A3 = B3;
    B1 = T1;
    B2 = T2;
    B3 = T3;
  }
  return sRet;

RSA 빠른 암호화, 복호화 알고리즘#

/*
    input: 평문 또는 암호문
    key: e 또는 d
    n: n
    output: 암호문 또는 평문
*/
int fastExp(int input, int key, int n)
{
    int output = 1;
    while(key) {
        if(key % 2 == 1) 
            output = (output * input) % n;
        input = (input * input) % n;
        key >>= 1;
    }
    return output;
}
